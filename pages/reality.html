Privacy

隐私声明

月读空间是个人网站。站点只为注册、登录、内容发布、留言互动、访问统计和 Live2D 房间体验处理必要数据。

数据类型 使用目的 保存位置与说明
账号信息 用于注册、登录、用户中心展示与权限判断。 包括用户名、邮箱、加密后的密码、角色与创建时间。密码不会以明文保存。
文章与留言 用于展示投稿、评论、留言审核和站内互动。 公开发布的内容可能被其他访客看到；后台保留审核、管理和删除能力。
访问统计 用于了解页面访问趋势、维护站点稳定性。 以站点统计数据为主，不用于广告画像或跨站追踪。
房间本地设置 用于保存 Live2D 房间的模型位置、聊天历史、LLM/TTS 配置等个人体验设置。 这类数据主要保存在你的浏览器 localStorage 中。清理浏览器站点数据会删除它们。
第三方接口配置 用于用户自行配置房间聊天或语音服务。 请不要在公共设备保存 API Key。站点不会主动将你的密钥写入公开页面。
Rights

用户权利与处理方式

如果你希望访问、更正或删除与自己相关的内容，请通过上方联系方式提交请求，并尽量说明账号、页面链接和需要处理的范围。

访问与更正

  • 登录后可在用户中心查看基础账号信息。
  • 发现公开内容有误时，可以提供链接申请更正。

删除与撤回

  • 你可以申请删除自己发布的留言、投稿或账号相关数据。
  • 浏览器本地房间设置可通过清理站点数据自行删除。

安全报告

  • 如发现 XSS、越权、敏感信息泄露等风险，请通过 GitHub Issues 或仓库联系方式报告。
  • 报告时请避免公开真实密钥、密码、令牌和他人隐私。
Notice

责任边界

月读空间包含虚拟角色、用户生成内容和可选第三方接口能力。现实边界写在这里，方便每位访客理解站点如何工作。

虚拟角色内容：Live2D 房间中的对话与语音体验可能来自用户配置的模型或第三方服务，内容仅用于互动体验，不构成专业建议。

外部链接：站点可能包含 GitHub、友链或用户提交链接。访问外部网站时，请阅读对方的隐私与安全政策。
