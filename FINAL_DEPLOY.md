## 🚀 三种部署方法

### 方法 1: 使用在线 SFTP 工具（最简单）

1. 访问：https://app.rclone.org/ 或类似在线 SFTP 工具
2. 配置连接：
   ```
   协议：SFTP
   主机：112.124.111.228
   端口：22
   用户名：root
   密码：@Aa620880123
   远程路径：/var/www/html/
   ```
3. 上传 `i18n.js` 和 `editor.html`

---

### 方法 2: 使用手机 SSH 应用

**Android**: JuiceSSH, Termux
**iOS**: Termius, Prompt

```bash
# 连接后执行
cd /var/www/html

# 使用 cat 创建文件（复制 i18n.js 内容粘贴）
cat > i18n.js

# 按 Ctrl+D 保存
```

---

### 方法 3: 通过已有网站后台

如果网站有文件管理功能：
1. 登录网站后台
2. 找到文件管理器
3. 上传到 `/var/www/html/`
