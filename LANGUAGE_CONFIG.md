# 🌙 月读空间 - 语言配置说明

## ✅ 语言配置已调整

项目已调整为**以中文为基本语言**，日文作为可选切换语言。

---

## 📋 配置详情

### 默认语言
- **语言**: 中文 (zh-CN)
- **HTML lang 属性**: `<html lang="zh-CN">`
- **localStorage 默认值**: `zh`

### 可选语言
- **语言**: 日文 (ja)
- **切换方式**: 点击右上角语言切换器

### 1. editor.html
```html

<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <title>新建投稿 | 月读空间</title>
```

**默认内容**:

### 3. stage.html
```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <title>主舞台 | 月读空间</title>
```

**默认内容**:
- 标题：主舞台
- 副标题：演唱会场馆 | 博客文章
- 筛选：全部、公告、传说、技术、其他
- 按钮：新建投稿

### 4. i18n.js
```javascript
const i18n = {
    zh: {  // 中文（默认）
        'login': '登录',
        'register': '注册',
        // ... 109 条翻译
    },
    ja: {  // 日文（切换）
        'login': 'ログイン',
        'register': '新規登録',
        // ... 109 条翻译
    }
};

// 默认语言
function getCurrentLang() {
    return localStorage.getItem('tsukuyomi_lang') || 'zh';  // 默认中文
}
```

## 🔄 语言切换

### 切换方式
1. 访问任意页面
2. 点击右上角语言切换器
3. 选择"日本語"
4. 界面立即切换为日文
5. 语言偏好自动保存

### 切换器位置
```
页面右上角:
┌─────────────────────┐
│ [中文] [日本語] │  ← 默认选中"中文"
└─────────────────────┘
```
